Personnalisation du Démineur Deluxe

Pour personnaliser le champ de mines

  1. Dans le menu Partie, cliquez sur Personnaliser.
  2. Personnalisez la taille du champ de mines en indiquant le nombre de carrés à afficher à la verticale et à l'horizontale et indiquez le nombre de mines à placer dans le champ.

Pour personnaliser le style de graphisme

  1. Dans le menu Options, cliquez sur Graphime.
  2. Choisir le répertoire contenant les images. Si l'une d'entre elle manque ou le nom est erroné, une erreur sera signalée

Pour afficher les statistiques

  1. Dans le menu Options, cliquez sur Statistique.
  2. Sélectionnez l'onglet du groupe voulu. Cochez ou décochez la case correspondantes au nom des membres dont vous calculer les statistiques.
