Vue d'ensemble du Démineur

But du jeu

Le but du jeu du Démineur est de localiser toutes les mines aussi rapidement que possible, sans en afficher aucune. Si vous découvrez une mine, vous avez perdu.

Pour jouer au Démineur

  1. Dans le menu Partie, cliquez sur Nouvelle.
  2. Cliquez sur l'un des carrés du champ de mines pour démarrer le chronomètre.

 Remarques :

  • Pour afficher le contenu d'un carré, cliquez dessus. Si vous découvrez une mine, vous avez perdu.
  • Si un nombre apparaît dans un carré, il indique le nombre de mines dissimulées dans les huit carrés qui l'entourent.
  • Pour marquer un carré qui selon vous contient une mine, cliquez dessus avec le bouton droit.
  • L'aire de jeu du Démineur comprend le compteur de mines, le chronomètre et le champ de mines.

Stratégies et astuces

  • Pour marquer un carré sur lequel vous avez des doutes, cliquez deux fois dessus avec le bouton droit. Un point d'interrogation (?) apparaît alors pour le marquer. Par la suite, vous pourrez soit marquer ce carré comme étant une mine, soit supprimer la marque en cliquant dessus une ou deux fois avec le bouton droit.
  • Si vous avez marqué toutes les mines qui entourent un carré portant un nombre, vous pouvez afficher le contenu des carrés adjacents restants en cliquant simultanément sur ce carré avec les boutons gauche et droit de la souris. Si toutes les mines qui entourent ce carré n'ont pas été marquées, les carrés adjacents restants non marqués seront enfoncés (ou clignoteront) lorsque vous cliquerez simultanément sur ce carré avec les boutons gauche et droit de la souris.
  • Recherchez des schémas communs dans les nombres. Souvent, mais pas toujours, un schéma de nombre signale un schéma de mines particulier. Par exemple, le schéma 2-3-2 placé au bord d'un groupe de carrés dont le contenu est affiché signale une rangée de trois mines placée à côté des trois nombres.
